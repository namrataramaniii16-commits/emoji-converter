# Emoji Converter

A beginner-friendly Python project that replaces simple text symbols with emojis.

## How It Works

The user enters a message, and the program replaces matching symbols with emojis.

Current emoji shortcuts:

- `:)` becomes `😊`
- `):` becomes `😔`

## How to Run

Make sure Python is installed on your computer.

Run the program with:

```bash
python main.py
```

On some systems, you may need:

```bash
python3 main.py
```

## Example Output

```text
> Hello :)
Hello 😊
```
